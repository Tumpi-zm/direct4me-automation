# Direct4me Service Portal – Playwright Automation PoC

This repository contains a small Playwright automation proof of concept prepared as a supplement to the Direct4me QA assignment.

The main assignment deliverable is the manual QA report. This automation project is included only as an additional demonstration of how manual testing coverage could start being converted into repeatable smoke/regression checks.
